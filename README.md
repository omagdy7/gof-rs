# Controls
* a: play animation
* n: next generation
* s: stop
* j or down arrow: go down next pattern (note: you have to stop the animation to browse the patterns)
* k or up arrow: go up next pattern

# How to run
```
git clone https://github.com/omagdy7/gof-rs
cd gof-rs
cargo run --release
```
